# benin_hackathon_huggingface_training_yor_eng

Note : The link to this repo is in the readme of submitted github repo link event on website.

This repo contains the .ipynb file which is used for training the model for our downstream task. and exporting it on drive. (deployed on hugging face)

Benin Multi Modal Hackathon 2024 13-16 May.

85% accuracy on test set (calculated just by matching strings of original and target text)
30 epochs
32 batch size
0.00001 learning rate
1300 parallel yoruba to english sentences
